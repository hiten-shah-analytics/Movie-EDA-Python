# 🎬 Movie Industry EDA — Python
Exploratory Data Analysis on 7,500+ movies spanning 4 decades.
## Key Findings
- Budget has strongest correlation with Gross Revenue (r = 0.74)
- Votes (audience engagement) is second strongest predictor (r = 0.63)
- IMDb Score has surprisingly weak correlation with Gross (r = 0.19)
- High ratings ≠ high earnings — quality alone doesn't guarantee box office success
## Tools Used
- Python, Pandas, NumPy, Seaborn, Matplotlib
## Techniques
- Data Cleaning & Type Conversion
- Missing Value Handling
- Correlation Matrix (Pearson, Kendall, Spearman)
- Scatter Plot & Regression Plot
- Full Numerized Heatmap (all features including categorical)
